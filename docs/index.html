Minimal Debian/Ubuntu Linux SD card images

Unoffical, minimal, pure and up-to-date vanilla Debian/Ubuntu Linux SD card images for various ARM-based single board computers.

Download

Minimal image

The images contains just enough to boot up. Great for development, testing and learning.

Pure image

The images includes only official Debian/Ubuntu packages, including the official Debian/Ubuntu Linux kernel, downloaded from the offical Debian/Ubuntu mirrors.

Up-to-date image

The images are rebuilt once per week to include the latest versions and security patches for each of the Debian/Ubuntu suites.
